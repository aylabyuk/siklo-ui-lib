## Accessibility

Every component meets these requirements:

- Full keyboard navigation
- Correct ARIA roles and attributes
- Focus management (trapping, restoration)
- Screen-reader announcements
- Storybook a11y addon audit with zero violations

## Token Coverage

All components use design tokens exclusively — no hardcoded values for:

- Colors
- Spacing
- Typography
- Border radius
- Shadows
- Motion (durations, easings)
